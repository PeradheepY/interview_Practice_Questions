## ğŸ” Code Flow (Logic Steps)

1. Start the program
2. Read the number of rows from the user
3. For each row `i` from 0 to `rows - 1`:
   - Print `i` spaces
   - Print `rows - i` stars
4. Move to the next line
5. End the program

---

## ğŸ’» C Code

```c
#include <stdio.h>

int main() {
    int rows, i, j, k;

    printf("Enter number of rows: ");
    scanf("%d", &rows);

    for(i = 0; i < rows; i++) {
        // Print spaces
        for(j = 0; j < i; j++) {
            printf("  ");
        }
        // Print stars
        for(k = 0; k < rows - i; k++) {
            printf("* ");
        }
        printf("\n");
    }

    return 0;
}
